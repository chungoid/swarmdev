Create a Python command-line password generator tool with the following features:

1. Generate random passwords with customizable length
2. Options for including/excluding uppercase, lowercase, numbers, and symbols
3. Ability to generate multiple passwords at once
4. Save generated passwords to a file (optional)
5. Command-line interface with proper argument parsing
6. Password strength indicator

The tool should be installable via pip and include proper documentation.
Target Users: Developers and security-conscious users who need secure passwords
Success Criteria: Users can quickly generate secure passwords with various options from the command line.
