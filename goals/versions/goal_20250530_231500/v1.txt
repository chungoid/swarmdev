Create a Go command-line file organizer tool with the following features:

1. Scan directories and organize files by type (images, documents, videos, etc.)
2. Configurable organization rules via YAML config file
3. Dry-run mode to preview changes before executing
4. Progress bar for long operations
5. Command-line interface with flags for source/destination directories
6. Logging of all file operations

The tool should follow Go best practices with proper module structure and be installable via go install.
Target Users: System administrators and power users who need to organize large file collections
Success Criteria: Users can efficiently organize thousands of files with customizable rules and safety features.
